# Robustness and Validation

## Sensitivity checks
The thesis tests whether the main findings hold when damping strength changes. If the results are stable across a range of damping scales, the interpretation is stronger.

## Bootstrap perturbation check
A bootstrap-style perturbation of the technical coefficient matrix A helps test whether the results are overly dependent on one exact network realization. If the core rankings persist under small perturbations, the result is more defensible.

## What robustness should show
A good robustness story should show:
- the direction of the main effect stays the same
- the magnitude changes smoothly rather than erratically
- the scenario ranking remains broadly consistent

## Validation limits
These checks do not prove the model is true. They test whether the results are fragile to plausible perturbations. That is the right standard for an empirical thesis of this kind.
